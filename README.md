# Super-Store-Retail-Analysis
Dashboard de Business Intelligence en Power BI para analizar KPIs de ventas, beneficios y tendencias de envío en un retail global.

Este es un proyecto de práctica utilizando el conocido dataset "Super Store". El objetivo fue crear un tablero en Power BI que permita ver rápidamente no solo cuánto vende la tienda, sino analizar la rentabilidad.

## ¿Qué quise resolver?
Diferenciar entre Ingresos (Ventas) y Beneficio (Profit).
Identificar qué productos se venden con pérdidas debido a descuentos excesivos.

## Herramientas usadas
Power BI: Para armar el reporte y los gráficos.
Power Query: Para revisar que los datos estuvieran limpios y correctos antes de empezar.
DAX Básico/Intermedio:Creación de medidas para calcular márgenes de ganancia y porcentajes.

## Insights Principales
Aunque la subcategoría "Tables" tiene muchas ventas, genera pérdidas importantes. Es un producto crítico que necesita revisión.
Descuentos peligrosos: Noté que cuando los descuentos superan el 20%, la rentabilidad cae drásticamente.
